In the asr folder, we use the local recognition sdk from Tencent with the original speech-recogniton import type code.
In the common folder, we use the log table to log into Tencent cloud automaticlly.
In the tts folder, we write the text to speech code. Although we use it on the local, it could be used in the catbox function.
The main speech recognition code is based on those SDK and lib, it finish all the speech recognition function.
