body_reco folder contains auxiliary code for body detection
face_reco folder contains auxiliary code for face recognition
gesture_reco folder contains auxiliary code for face recognition
speech_reco folder contains auxiliary code for face recognition
resources folder contains game background music files

all the auxiliary functions are called by vh_demo.py, which is the main file for python server.

\\To run the server, please ensure run the code in aconda enviroment.
\\To load the enviroment in the  SRC package, plaese use the follow code:

conda env update -f=environment.yml

\\or

pip install -r requirements.txt


Running command:
python vh_demo.py
